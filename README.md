# MyFirstPrpject

A simple Library Management System in Java.

## Features (to be added)
- Add books
- View books
- Issue books
- Return books

## How to run
1. Compile the Java file:
   ```
   javac src/LibraryManagementSystem.java
   ```
2. Run the program:
   ```
   java -cp src LibraryManagementSystem
   ```
